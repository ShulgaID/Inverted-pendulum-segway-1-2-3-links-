# Inverted-pendulum-segway-1-2-3-links-
Проект по дисциплине "Программированию роботов" (осень 2025)

# Цель
Разработать модульную, хорошо документированную симуляцию системы "Перевернутый маятник" (1, 2 и 3 звена) в физическом движке MuJoCo и реализовать алгоритмы стабилизации (удержания в вертикальном положении) для демонстрации навыков моделирования, управления и программной инженерии.

# Задачи
1. Создать XML-модели в MuJoCo для трех конфигураций:
   - Однозвенный маятник (Cart-Pole): Тележка на рейке с одним маятником.
   - Двухзвенный маятник (Double Pendulum): Аналог Segway или "палка на палке".
   - Трехзвенный маятник (Triple Pendulum): Усложненная неустойчивая система.
2. Реализовать классические алгоритмы управления:
   - Линеаризация и Linear Quadratic Regulator (LQR).
   - Proportional-Integral-Derivative (PID) контроллер.
3. Обеспечить визуализацию процесса симуляции и графиков состояния системы.
4. Проектировать код согласно принципам, использовать подходящие паттерны проектирования.
5. Полностью задокументировать проект.
