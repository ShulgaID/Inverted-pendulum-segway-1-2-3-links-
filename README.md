# Inverted-pendulum-segway-1-2-3-links

Проект по дисциплине "Программирование роботов" (осень 2025)

---

## Цель проекта

Разработать **модульную, расширяемую и документированную симуляцию системы «перевёрнутый маятник»**  
в физическом движке **MuJoCo**, а также реализовать и сравнить
классические алгоритмы стабилизации для удержания системы в вертикальном положении.

Проект направлен на демонстрацию навыков:
- физического моделирования,
- теории управления,
- архитектурного проектирования программ,
- работы с симуляторами робототехнических систем.

---

## Задачи проекта

1. Создать XML-модель перевёрнутого маятника в MuJoCo.
2. Реализовать алгоритмы управления:
   - **PID-регулятор** (с весами и анти-windup),
   - **LQR-регулятор** на основе линеаризованной модели.
3. Обеспечить:
   - визуализацию симуляции в MuJoCo Viewer,
   - сбор и сохранение данных состояния системы.
4. Спроектировать программную архитектуру в соответствии с принципами:
   - SOLID,
   - DRY,
   - KISS,
   - YAGNI.
5. Полностью задокументировать код и архитектурные решения.
